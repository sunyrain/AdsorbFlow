# Documentation

This folder contains the public documentation for the AdsorbFlow release.

| File | Purpose |
|---|---|
| `DATA.md` | Dataset, checkpoint, and artifact placement |
| `REPRODUCIBILITY.md` | End-to-end reproduction workflow |
| `COMMANDS.md` | Path-agnostic command templates |
| `PAPER_RESULTS.md` | Manuscript result tables and audit notes |
| `MODEL_CARD.md` | Intended use, limitations, and evaluation scope |
| `REPOSITORY_STRUCTURE.md` | Directory-level repository map |

Large datasets, checkpoints, generated trajectories, VASP outputs, and private
cluster paths are intentionally excluded from Git.
